This project is a simple yet visually meaningful web-based calendar created using HTML and JavaScript. It displays all twelve months of the year in a clean vertical layout while incorporating a color theme inspired by the Indian national flag. The primary objective of the project is to demonstrate basic DOM manipulation and styling techniques in JavaScript while also presenting an aesthetically organized representation of months.

The structure of the project begins with standard HTML boilerplate code, including the document type declaration and meta tags for character encoding and responsiveness. Inside the body, a main container `<div>` with the class `year` is used to hold all the months. Each month is represented by its own `<div>` element with the class `month`, and the names of the months are written in uppercase, ranging from January to December. This creates a simple and readable layout that is easy to style and manipulate.

The styling and dynamic behavior of the calendar are handled using JavaScript. Instead of relying on external CSS, the script directly modifies the styles of the elements using DOM methods. First, the script selects the parent container using `document.querySelector('.year')`. It then iterates over all child elements (the months) using a loop. In the first loop, padding is applied uniformly to each month to improve spacing and readability.

Next, the project uses multiple loops to divide the months into three groups of four and apply background colors accordingly. The first four months (January to April) are given an orange background, the next four months (May to August) are styled with a white background, and the final four months (September to December) are assigned a green background. These three colors collectively resemble the Indian tricolor flag, giving the project a thematic and cultural significance.

In addition to the background colors, another loop is used to make all the month names bold by setting the `fontWeight` property to `"bold"`. This enhances the visibility of the text and ensures consistency across all elements.

Overall, this project effectively demonstrates how JavaScript can be used to dynamically style HTML elements without external CSS. It highlights key concepts such as element selection, iteration through child nodes, and inline styling. The use of loops to categorize and style elements makes the code efficient and easy to understand. While the design is simple, the project successfully combines functionality with a meaningful visual theme, making it a good example for beginners learning web development and DOM manipulation.
